## What to Test Per Role

### Superadmin / Admin
- [ ] Login and access admin dashboard
- [ ] View all users list
- [ ] View pending verifications
- [ ] View monitoring map (all users)
- [ ] Access prescription overview

### Doctor
- [ ] Login and see doctor dashboard
- [ ] View own appointments (confirmed, pending, completed)
- [ ] Update appointment status (confirm / complete / no-show)
- [ ] View patient prescriptions
- [ ] Appear on map (verified = true)

### Pharmacy
- [ ] Login and see pharmacy dashboard
- [ ] Appear on map (verified = true)
- [ ] View own profile and services
- [ ] Check ratingSummary is populated

### Patient
- [ ] Login and see patient dashboard
- [ ] View own prescriptions
- [ ] Book appointment with a doctor from the map
- [ ] View and cancel pending appointments
- [ ] Submit rating for a doctor or pharmacy
